# (C# 코딩) 로그인

## 개요
- **C# 프로그래밍 학습**: Windows Forms를 이용한 사용자 인터페이스(UI) 설계 및 이벤트 기반 프로그래밍 기초 습득
- **1줄 소개**: 사용자 아이디와 비밀번호를 입력받아 일치 여부를 확인하고 메시지를 출력하는 로그인 프로그램
- **사용한 플랫폼**: C#, .NET Windows Forms, Visual Studio, GitHub
- **사용한 컨트롤**: Label, TextBox, Button
- **사용한 기술과 구현한 기능**:
    - **이벤트 핸들러**: `Enter`, `Leave` 이벤트를 활용한 텍스트박스 힌트(Placeholder) 및 글자색 변경 기능
    - **조건문(if-else)**: 입력된 문자열과 미리 설정된 계정 정보(`myID`, `myPW`) 비교 로직 구현
    - **메시지 박스**: `MessageBox`를 활용하여 로그인 성공/실패 시 사용자에게 시각적 피드백 제공
    - **형상 관리**: GitHub를 통한 프로젝트 버전 관리 및 충돌 해결 과정 실습

## 과제 내용
1. **텍스트박스 힌트 제어**: 
   - 포커스 진입 시: 안내 문구 삭제 및 글자색 `Black` 변경
   - 포커스 해제 시: 빈 칸일 경우 안내 문구 복구 및 글자색 `Gray` 변경
2. **로그인 유효성 체크**: 
   - 설정 아이디: `1234` / 설정 비밀번호: `yuiop`
   - `&&` 연산자를 사용하여 두 조건이 모두 충족될 때만 성공 처리
3. **사용자 경험(UX) 개선**: 
   - 로그인 실패 시 비밀번호 입력창 초기화(`Clear`) 및 포커스 자동 이동

## 배운 내용
- **이벤트 기반 프로그래밍**: `Click`, `Enter`, `Leave` 등 다양한 사용자 액션에 따른 코드 실행 원리를 이해함
- **변수와 데이터 타입**: `string` 변수를 선언하여 고정된 계정 정보를 관리하고 입력값과 비교하는 방법을 익힘
- **Git 형상 관리**: 웹과 로컬 저장소 간의 싱크 오류로 인한 충돌(Conflict) 해결 방법을 학습하고 리포지토리 재설정 과정을 실습함
- **디버깅 및 로직 검증**: 조건문을 통해 예상치 못한 입력값에 대한 예외 상황을 처리하는 중요성을 배움
